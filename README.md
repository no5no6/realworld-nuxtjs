# realworld-nuxtjs
在线 [Demo](http://realworld.no5no6.com/)  
nuxt 的练习项目。基于 [realworld](https://github.com/gothinkster/realworld) 项目构建。

## 安装与使用
  + 安装依赖 `yarn install`
  + 运行 `yarn serve`
  + 访问 [localhost:3000](http://localhost:3000)
